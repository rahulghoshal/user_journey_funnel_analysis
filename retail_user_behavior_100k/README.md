## The Retail AI Shift: 100K Sessions Decoded

What actually drives a customer to purchase — and where do they drop off?

---

## What’s inside

108,000+ context-rich interaction events generated to simulate real-world e-commerce systems.

Each row represents a user interaction within a session, including:

- Session_ID — full journey tracking  
- User_ID — synthetic user identifier  
- User_Action — view, click, wishlist, add-to-cart, purchase, drop  
- Product metadata — category, brand, price  
- Timestamp — sequence-aware behavioral data  
- Channel & device — web, mobile, app usage patterns  
- Region & traffic source — global retail simulation  

Every record is structured for meaningful analysis — no low-signal or empty data.

## Pre-engineered ML features

This dataset is designed for immediate use in machine learning workflows:

- is_conversion — purchase prediction label  
- drop_off_flag — identifies abandoned sessions  
- session_length — total events per session  
- interaction_count — position within session  
- time_spent_sec — engagement signal  

No heavy preprocessing required.
